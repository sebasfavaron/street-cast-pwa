# Device PWA Specification

## Overview

A Progressive Web App (PWA) that runs on Raspberry Pi devices to display advertising content. The PWA polls the server for video manifests, downloads videos, and plays them in a loop while reporting impressions.

## Technical Requirements

### Core Functionality

- **Manifest Polling**: Request video playlist from `/api/manifest/[deviceId]` every 2-3 minutes
- **Video Management**: Download MP4 files and cache them locally
- **Playback Loop**: Play videos continuously in sequence
- **Impression Tracking**: Send POST to `/api/impression` when each video starts playing
- **Offline Resilience**: Continue playing cached videos if server is unreachable

### Device Requirements

- **Raspberry Pi** with internet connection
- **Chromium browser** in kiosk mode
- **Local storage** for video caching (minimum 2GB recommended)
- **Display**: Any HDMI-compatible screen

## Implementation Details

### Manifest Polling

```typescript
interface ManifestResponse {
  version: string;
  deviceId: string;
  creatives: Creative[];
  generatedAt: string;
}

interface Creative {
  id: string;
  url: string;
  duration: number;
  campaignId: string;
  campaignName: string;
}

class ManifestPoller {
  private readonly POLL_INTERVAL = 2 * 60 * 1000; // 2 minutes
  private readonly DEVICE_ID = process.env.DEVICE_ID || 'device_123';

  async pollManifest(): Promise<void> {
    try {
      const response = await fetch(`/api/manifest/${this.DEVICE_ID}`);
      const manifest: ManifestResponse = await response.json();
      await this.videoManager.updatePlaylist(manifest.creatives);
    } catch (error) {
      console.error('Failed to fetch manifest:', error);
      // Continue with cached videos
    }
  }
}
```

### Video Playback

```typescript
class VideoPlayer {
  private currentVideo: Creative | null = null;
  private playlist: Creative[] = [];
  private currentIndex = 0;
  private videoElement: HTMLVideoElement;

  constructor() {
    this.videoElement = document.createElement('video');
    this.setupVideoElement();
  }

  private setupVideoElement(): void {
    this.videoElement.autoplay = true;
    this.videoElement.muted = true;
    this.videoElement.loop = false;
    this.videoElement.addEventListener('ended', () => this.playNext());
    this.videoElement.addEventListener('play', () => this.onVideoStart());
  }

  async playNext(): Promise<void> {
    if (this.playlist.length === 0) return;

    const video = this.playlist[this.currentIndex];
    await this.loadVideo(video);
    this.currentIndex = (this.currentIndex + 1) % this.playlist.length;
  }

  private onVideoStart(): void {
    if (this.currentVideo) {
      this.impressionTracker.reportImpression(this.currentVideo.id);
    }
  }
}
```

### Caching Strategy

```typescript
// Using Dexie.js for IndexedDB management
import Dexie from 'dexie';

class VideoCache extends Dexie {
  videos!: Table<CachedVideo>;

  constructor() {
    super('VideoCache');
    this.version(1).stores({
      videos: 'id, url, blob, downloadedAt, expiresAt, size',
    });
  }
}

interface CachedVideo {
  id: string;
  url: string;
  blob: Blob;
  downloadedAt: number;
  expiresAt: number;
  size: number;
}

class CacheManager {
  private db = new VideoCache();
  private readonly MAX_CACHE_SIZE = 2 * 1024 * 1024 * 1024; // 2GB
  private readonly MAX_VIDEOS = 10;

  async cacheVideo(creative: Creative, videoBlob: Blob): Promise<void> {
    const cachedVideo: CachedVideo = {
      id: creative.id,
      url: creative.url,
      blob: videoBlob,
      downloadedAt: Date.now(),
      expiresAt: Date.now() + 24 * 60 * 60 * 1000, // 24 hours
      size: videoBlob.size,
    };

    await this.db.videos.put(cachedVideo);
    await this.cleanupCache();
  }

  private async cleanupCache(): Promise<void> {
    const videos = await this.db.videos.orderBy('downloadedAt').toArray();
    const totalSize = videos.reduce((sum, video) => sum + video.size, 0);

    if (totalSize > this.MAX_CACHE_SIZE || videos.length > this.MAX_VIDEOS) {
      const videosToDelete = videos.slice(0, videos.length - this.MAX_VIDEOS);
      await this.db.videos.bulkDelete(videosToDelete.map((v) => v.id));
    }
  }
}
```

**Caching Features:**

- Cache videos in IndexedDB with expiration timestamps
- Keep last 10 videos cached with 2GB storage limit
- Automatic cleanup of old videos when storage is full
- Pre-download next video while current is playing
- Workbox integration for advanced caching strategies
